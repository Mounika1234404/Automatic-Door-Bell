# Automatic Doorbell System (C Language)

## Overview

This project implements an **Automatic Doorbell System** using the C programming language in an online simulator.
The system automatically detects the presence of a visitor using simulated sensor input and triggers a doorbell alert.

The project demonstrates basic concepts of **embedded systems, sensor-based automation, and event-driven programming**.

---

## Features

* Automatic visitor detection using simulated sensor input
* Doorbell alert triggered automatically
* Simple logic implemented in C
* Demonstrates basic automation concepts

---

## Technologies Used

* **C Programming Language**
* **Online Embedded Systems Simulator**

---

## How the System Works

1. The system continuously monitors sensor input.
2. When a visitor is detected near the door, the sensor sends a signal.
3. The program processes the signal.
4. The doorbell is automatically triggered to notify the house owner.

---

## How to Run the Project

1. Open the online simulator used for the project.
2. Copy and paste the C program into the simulator editor.
3. Compile the program.
4. Run the simulation to observe the automatic doorbell behavior.
